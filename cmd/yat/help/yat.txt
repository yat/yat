Yat is a message server.

Usage:

  yat COMMAND ARGS

Commands:

  publish     publish a message
  subscribe   print selected messages
  request     publish a request
  respond     respond to requests
  serve       run a server instance
  seed        generate local creds

Client Flags:

  -server localhost:25120 ($YAT_SERVER)
      Address of the Yat server.
      The default port is 443.
  
  -token-file PATH ($YAT_TOKEN_FILE)
      Path to a file containing a JWT signed by a trusted issuer.
      The client reads this file every time it dials.
      This flag is ignored if $YAT_TOKEN is set.

Shared Flags:

  -tls-dir PATH ($YAT_TLS_DIR)
      Path to a directory containing tls.crt and tls.key files.
      The directory may also contain a ca.crt file for verification.

  -log-level info ($YAT_LOG_LEVEL)
      Sets the minimum log level.
      Values are debug, warn, info, and error.
