This is a command-line client and server for Yat, a message bus.
Learn more about Yat by visiting yat.io or github.com/yat/yat.

Usage:

  yat command [args]

Commands:

  publish     publish a message
  subscribe   receive messages
  request     publish a request
  respond     respond to requests
  serve       run the server
  local       generate local config
  help        print command help
  version     print Yat version

Run 'yat help command' for more information about a command.

Shared flags:

  -addr localhost:63197 ($YAT_ADDR)
      TCP address of the server.

  -log-level error ($YAT_LOG_LEVEL)
      Set the log level to debug, info, warn, or error.
      Logs are printed to stderr as JSON lines.

  -tls-cert-file PATH ($YAT_TLS_CERT_FILE)
      A file containing a PEM-encoded x.509 certificate.
      If this flag is set, -tls-key-file must be set too.

  -tls-key-file PATH ($YAT_TLS_KEY_FILE)
      A file containing the corresponding PEM-encoded private key.
      If this flag is set, -tls-cert-file must be set too.

  -tls-ca-file PATH ($YAT_TLS_CA_FILE)
      A file containing PEM-encoded root certificates.

  -local DIR ($YAT_LOCAL_DIR)
      Load credentials and configuration from this directory,
      overriding any other flags or environment variables.
      See "yat help local" for details.
