Respond publishes responses to a stream of matching requests.
It reads static response data from stdin by default.

Usage:

  yat respond [flags] PATTERN

Flags:

  -file, -f /dev/stdin
      A file of response data.

  -empty, -e
      Publish empty responses.

  -limit, -n 0
      Unsubscribe after n responses.

  -duration 0s
      Unsubscribe after this much time.
      Duration is expressed as Nunit, like
      1ms, 1s, 1h. The largest unit is h.
