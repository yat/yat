Serve runs an instance of the Yat server.

Usage:

  yat serve [flags]

Flags:

  -bind HOST:PORT
      Local TCP address for the server to announce.
      If this flag is not set, the server announces the address
      set by the YAT_ADDR env var or the -addr shared flag.

  -local-tls DIR
      Generate and use a directory of local TLS credentials,
      including a root CA certificate and client/server keypairs.
      The directory is recreated if any files are missing,
      or if the server's address changes.
