Serve runs a Yat server.

Usage:

  yat serve [flags]

Flags:

  -auth FILE
      A JSON or YAML file containing client auth rules.
      If the -local flag is set, auth rules are loaded
      from the local rules.yaml instead of this file.

  -bind HOST:PORT
      Local TCP address for the server to announce.
      If this flag is not set, the server announces the address
      set by the YAT_ADDR env var or the -addr shared flag.
      The default address is localhost:63197.
