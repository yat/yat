Serve runs a Yat server.

Usage:

  yat serve

Flags:

  -bind localhost:25120
      The address the server should bind.

  -config FILE
      Path to a JSON or YAML configuration file.
      This flag can be set more than once.

  -tls-require-client-cert true
      If the value of this flag is true (the default),
      clients must present a verifiable certificate.

The server requires a TLS certificate and key to be configured.
