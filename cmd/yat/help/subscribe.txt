Subscribe prints a stream of selected messages.
Messages are printed as JSON objects, one per line.

Usage:

  yat subscribe PATH

Flags:

  -group, -g NAME
      Add the subscriber to a route group.
      Messages are routed to a random group member.

  -limit, -n N
      Unsubscribe after N deliveries.

  -duration 0s
      Unsubscribe after some amount of time.

  -raw
      Print raw message data.
