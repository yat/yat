Seed generates local credentials for development and testing.

Usage:

  yat seed DIR

Generated Files:

  tls/client.crt
      PEM-encoded x.509 client certificate.

  tls/client.key
      PEM-encoded client private key.

  tls/server.crt
      PEM-encoded x.509 server certificate.

  tls/server.key
      PEM-encoded server private key.

  tls/ca.crt
      PEM-encoded signing certificate.
